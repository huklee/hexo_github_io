# Huklee's Vim cheet sheet

## 1. Using bash shell commands

  ``` bash
  :! [bash_cmd]
  ```

  - run a shell
  ``` bash
  :sh
  ```

## 2. Auto-indent

  - all lines
  ``` vim
  gg=G
  ```

  - for selected lines
  ``` vim
  [selected mode]
  =
  ```
